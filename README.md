# ✈️ Flight Management System

A professional, modern, and minimalist web application designed to streamline flight scheduling and quotation management. Built with a focus on clean UI/UX and efficient data handling, this system is a robust tool for private aviation and travel agencies.

![Flight Management Header](/assets/image.png)

## 🚀 Key Features

### 📊 Intelligent Dashboard
- **Real-time Metrics**: Instant overview of today's flights, total operations, and pending confirmations.
- **Smart Filtering**: Advanced filtering system by date, airport code (CHM, PVR, GDL, ZLO), and confirmation status.
- **Interactive Data Views**: Cleanly organized tables with quick-edit actions.

### 💰 Advanced Quotation Engine
- **Multi-passenger Support**: Dynamic interface to add/remove passengers within a single quote.
- **Variable Pricing**: Automated calculation based on service type (Private Aviation, Shared Shuttle, Suite) and aircraft model.
- **Extras Management**: Real-time addition of luxury services (Veuve Clicquot, Excess Weight, etc.) with automatic total synchronization.

### 📝 Operations Management
- **Flight Itineraries**: Detailed flight data capture including load factor, trip sheets, and airport origin.
- **Data Persistence**: Reliable local data storage using `LocalStorage`, ensuring no mission-critical info is lost.
- **One-Click Exports**: Professional PDF generation for both flights and quotes, ready for client delivery.

### 🎨 Modern Design System
- **Teal Identity**: Sophisticated corporate color palette (`#004B4B`).
- **Typography**: Optimized readability with the `Inter` font family.
- **Responsive Layout**: Fluid experience across mobile, tablet, and desktop devices.
- **Interactive UX**: Smooth micro-animations and intuitive navigation feedback.

## 🛠️ Built With

*   **Logic**: JavaScript (ES6+)
*   **Structure**: Semantic HTML5
*   **Styling**: CSS3 (Custom Design System) & Bootstrap 5
*   **PDF Generation**: [jsPDF](https://github.com/parallax/jsPDF)
*   **Icons**: FontAwesome 6
